# Aura Roll — Лотерея Аур 🔮

**Aura Roll** — это стильная и захватывающая веб-игра в жанре RNG (генератор случайных чисел), вдохновленная популярными режимами из Roblox. Испытайте свою удачу, прокручивая лотерею аур, собирая редкие визуальные эффекты, улучшая свои показатели в магазине и открывая новые уникальные косметические скины!

## ✨ Особенности игры
1. **Коллекция аур (от 1/2 до 1/100)**:
   - **1/2 — Обычная аура**: Легкое серое облако пыли вокруг персонажа.
   - **1/4 — Необычная аура**: Спокойное зеленое облако с взлетающими листьями.
   - **1/8 — Редкая аура**: Яркое синее облако с радиальными импульсами под ногами.
   - **1/16 — Эпическая аура**: Фиолетовые сияющие звездочки и магический фиолетовый туман.
   - **1/32 — Воздушная аура**: Закручивающиеся спирали ветра вокруг тела и анимированная тучка над головой, пускающая потоки воздуха.
   - **1/64 — Водная аура**: Падающий водопад капель по телу персонажа и расходящиеся волны на платформе.
   - **1/100 — Легендарная аура**: Огромная сияющая золотая звезда на груди, столпы восходящего света и густой солнечный туман.
2. **Проработанный Canvas-движок**:
   - Плавная отрисовка 3D-платформы и анимированного человечка с idle-анимацией (дыхание, покачивание рук).
   - Индивидуальные физические частицы для каждого типа ауры.
   - Динамический прыжок (бэкфлип) человечка во время крутки.
   - Эффект тряски экрана при выпадении редких аур.
3. **Экономика и Прогрессия**:
   - За каждую крутку начисляются **Монеты Аур**. Чем реже аура, тем больше монет она приносит!
   - **Магазин улучшений**:
     - *Клевер Удачи* (повышает шанс выпадения редких аур).
     - *Быстрый Спиннер* (сокращает время перезарядки крутки).
     - *Зелье Фортуны* (временно удваивает удачу на 60 секунд с возможностью стака времени).
4. **Скины и Кастомизация**:
   - Меняйте внешний вид платформы: Золотая, Кибер-Неон, Лава или Бездна!
   - Меняйте скины человечка: Король (с короной), Рыцарь (с плюмажем) или Тень (с темными частицами).
5. **Комфорт игрока**:
   - Авто-крутка (Auto Roll) для непрерывного процесса.
   - Быстрая крутка (Quick Roll) для мгновенного получения результатов без анимации.
   - Интерактивная «Книга Аур» для просмотра собранной коллекции и экипировки аур.
   - Двуязычный интерфейс (Русский / English) с кнопкой переключения.
   - Встроенный синтезатор звуков (Web Audio API) для атмосферных щелчков, спинов и триумфальных звуков при выпадении легендарных предметов.
   - Автоматическое сохранение прогресса в LocalStorage.

## 🛠️ Структура проекта
- `index.html` — основная разметка интерфейса, сайдбаров и контейнера игрового поля.
- `styles.css` — премиальный адаптивный дизайн с использованием Glassmorphism-эффектов, градиентов и неонового свечения.
- `script.js` — игровая логика, математика вероятностей, синтезатор звуков, обработка магазинов и рендеринг частиц на Canvas.

## 🚀 Как запустить игру
Игра полностью автономна и не требует сборки.
1. Откройте файл `index.html` в любом современном браузере.
2. Либо запустите локальный сервер (например, с помощью расширения Live Server в VS Code или выполнив `python -m http.server 8000` в папке с игрой) и перейдите по адресу `http://localhost:8000`.
